# Literature References

This directory contains BibTeX citations for papers and resources used in this project.

## Files

- **levenshtein.bib** - Wagner & Fischer (1974): The foundational paper on string edit distance (Levenshtein distance)
- **ringer_2020_twitchchat.bib** - Ringer et al. (2020): TwitchChat dataset paper for understanding livestream chat patterns
- **ringer_2022_phd.bib** - Ringer (2022): PhD thesis on multi-modal livestream highlight detection
- **barbieri_2017_emotes.bib** - Barbieri et al. (2017): Study on modeling Twitch emotes and gaming audience language

## Usage

These citations can be imported into your LaTeX documents or reference managers. Each file contains a single entry for easy inclusion in your bibliography.
